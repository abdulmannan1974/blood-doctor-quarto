Observable Plot Master Showcase

Comprehensive exploratory data visualization — every mark, transform, facet, and interaction

TipThe Grammar of Graphics in the Browser

Observable Plot (Plot) is a declarative JavaScript library for exploratory data visualization. It combines composable marks, data transforms (binning, grouping, stacking, dodging, density), faceted small multiples, and interactive tooltips (Plot.tip()) into a concise grammar of graphics interface that executes natively in Quarto without server reliance.

1. Scatterplots & Beeswarm Dodge (Plot.dot, Plot.dodgeX)

Scatterplots assign numerical attributes to \(X\) and \(Y\) dimensions. When overplotting occurs, Plot.dodgeX() or Plot.dodgeY() arranges points into beeswarm distributions without overlap.

Code 
// Simulated patient sample dataset
patients = d3.range(120).map(i => {
  const genotype = ["HbSS", "HbS/β0", "HbS/β+", "HbSC"][i % 4];
  const sex = i % 2 === 0 ? "Female" : "Male";
  const hb = genotype === "HbSS" ? 6.5 + Math.random() * 2.5 :
             genotype === "HbS/β0" ? 7.2 + Math.random() * 2.2 :
             genotype === "HbS/β+" ? 9.0 + Math.random() * 2.5 :
             10.5 + Math.random() * 2.5;
  const hbf = genotype === "HbSS" ? 8 + Math.random() * 14 :
              genotype === "HbS/β0" ? 5 + Math.random() * 10 :
              1 + Math.random() * 5;
  const retic = 120 + (15 - hb) * 25 + (Math.random() - 0.5) * 40;
  return { id: `P-${100+i}`, genotype, sex, hb, hbf, retic };
})
Code 
Plot.plot({
  title: "Hemoglobin (Hb) vs Fetal Hemoglobin (HbF) by Genotype",
  width: 760,
  height: 380,
  grid: true,
  x: {label: "Hemoglobin Hb (g/dL) →"},
  y: {label: "↑ Fetal Hemoglobin HbF (%)"},
  color: {legend: true},
  marks: [
    Plot.ruleY([0]),
    Plot.dot(patients, {
      x: "hb",
      y: "hbf",
      fill: "genotype",
      r: 4.5,
      fillOpacity: 0.8,
      tip: true
    })
  ]
})
Code 
Plot.plot({
  title: "Beeswarm Distribution of Reticulocyte Count across Genotypes",
  width: 760,
  height: 280,
  x: {label: "Genotype"},
  y: {label: "Reticulocyte Count (x10⁹/L)", grid: true},
  color: {legend: true},
  marks: [
    Plot.dot(patients, Plot.dodgeX("dot", {
      x: "genotype",
      y: "retic",
      fill: "genotype",
      r: 4,
      tip: true
    }))
  ]
})

2. Histograms & Automated Binning (Plot.rectY, Plot.binX)

The Plot.binX() transform automatically bins continuous numerical data and counts occurrences:

Code 
Plot.plot({
  title: "Hemoglobin Frequency Histogram",
  width: 760,
  height: 320,
  grid: true,
  x: {label: "Hemoglobin Hb (g/dL)"},
  y: {label: "↑ Patient Count"},
  color: {legend: true},
  marks: [
    Plot.rectY(patients, Plot.binX({y: "count"}, {x: "hb", fill: "sex", thresholds: 15})),
    Plot.ruleY([0])
  ]
})

3. 2D Density Contours (Plot.density, Plot.contour)

Density contours visualize bivariate continuous distributions without clutter:

Code 
Plot.plot({
  title: "2D Density Estimation (Hb vs Reticulocytes)",
  width: 760,
  height: 360,
  grid: true,
  x: {label: "Hemoglobin Hb (g/dL)"},
  y: {label: "Reticulocytes (x10⁹/L)"},
  marks: [
    Plot.density(patients, {x: "hb", y: "retic", stroke: "genotype", strokeWidth: 1.5}),
    Plot.dot(patients, {x: "hb", y: "retic", fill: "genotype", r: 2, opacity: 0.5})
  ]
})

4. Grouped & Stacked Bar Charts (Plot.barY, Plot.groupX)

Categorical summaries aggregated using Plot.groupX():

Code 
Plot.plot({
  title: "Mean Hemoglobin Level by Genotype & Sex",
  width: 760,
  height: 320,
  x: {label: "Genotype"},
  y: {label: "Mean Hb (g/dL)", grid: true},
  color: {legend: true},
  marks: [
    Plot.barY(patients, Plot.groupX({y: "mean"}, {x: "genotype", y: "hb", fill: "sex", fx: "sex"})),
    Plot.ruleY([0])
  ]
})

5. Linear Regression & Trend Lines (Plot.linearRegressionY)

Statistical trendlines with automatically calculated confidence bands:

Code 
Plot.plot({
  title: "Linear Regression: Reticulocytes vs Hemoglobin",
  width: 760,
  height: 340,
  grid: true,
  x: {label: "Hemoglobin Hb (g/dL)"},
  y: {label: "Reticulocytes (x10⁹/L)"},
  marks: [
    Plot.dot(patients, {x: "hb", y: "retic", fill: "genotype", opacity: 0.6}),
    Plot.linearRegressionY(patients, {x: "hb", y: "retic", stroke: "#b71c1c", strokeWidth: 2.5, ci: 0.95})
  ]
})

7. Faceted Small Multiples (fx, fy)

Faceting creates sub-plots across multiple dimensions for deep cohort comparison:

Code 
Plot.plot({

  width: 760,
  height: 420,
  grid: true,
  x: {label: "Hb (g/dL)"},
  y: {label: "HbF (%)"},

    Plot.dot(patients, {
      x: "hb",
      y: "hbf",
      fill: "genotype",
      fx: "sex",
      fy: "genotype",

      tip: true
    })
  ]
})

9. Summary of Observable Plot Mark Primitives

Mark Primitive Visual Element Primary Clinical / Scientific Application
Plot.dot() Circle / Symbol Scatterplots, beeswarm plots, individual patient points
Plot.lineY() Polyline Time-series, pharmacokinetic clearance, survival curves
Plot.areaY() Polygon / Area Cumulative dose, confidence bands, streamgraphs
Plot.barY() Rectangle Categorical bar charts, count frequency
Plot.rectY() Binned Rect Continuous histograms, 2D heatmaps
Plot.boxY() Box & Whiskers Quartiles, median, IQR, statistical outlier detection
Plot.density() Contour Lines Bivariate kernel density estimates
Plot.linearRegressionY() Line + Band Trendlines with 95% confidence intervals
Plot.cell() Grid Cell Correlation matrices, categorical heatmaps
Plot.ruleY() Line Reference thresholds (e.g., MEC, normal ranges)
Plot.tip() HTML Callout Interactive mouse-over tooltips
