The Journey: 26 Years of Learning

When I look back, I remember creating my first PowerPoint presentation in the early 2000s on a hospital laboratory computer. It was full of mistakes, with poor formatting and probably the worst alignment imaginable. But it was a beginning.

Nearly 26 years later, I have completed my training in Quarto, an open-source scientific publishing system that brings writing, coding, data analysis, and publication into one reproducible workflow.

Using Quarto with R, Python, Jupyter, Julia, Node.js, and Pandoc, I can now create statistical figures myself—from survival curves and forest plots to publication-ready tables, mathematical equations, and interactive visualisations.

Demonstrating the Quarto Multi-Engine Toolset

1. R Engine ({r}): Kaplan-Meier Survival Curve

Using R and ggplot2, we generate a publication-ready Kaplan-Meier survival curve for clinical trials:

Code 
library(ggplot2)

set.seed(2026)
time <- seq(0, 36, by=1)
surv_control <- exp(-0.06 * time) + rnorm(length(time), 0, 0.01)
surv_treated <- exp(-0.025 * time) + rnorm(length(time), 0, 0.01)

df <- data.frame(
  Time = rep(time, 2),
  Survival = c(pmax(0, surv_control), pmax(0, surv_treated)),
  Arm = factor(rep(c("Standard Care", "Targeted Therapy"), each=length(time)))
)

ggplot(df, aes(x=Time, y=Survival, color=Arm, linetype=Arm)) +
  geom_line(linewidth=1.2) +
  scale_color_manual(values=c("#7f0000", "#2e7d32")) +
  scale_y_continuous(labels=scales::percent, limits=c(0, 1)) +
  labs(title="Overall Survival in Relapsed/Refractory Disease",
       subtitle="Kaplan-Meier Survival Estimates by Treatment Arm",
       x="Follow-up Duration (Months)", y="Overall Survival (%)") +
  theme_minimal(base_size=12) +
  theme(legend.position="bottom", plot.title=element_text(face="bold", color="#7f0000"))

As demonstrated in Figure 1, targeted therapy significantly extends overall survival compared to standard care.

2. R Engine ({r}): Meta-Analysis Forest Plot

Forest plots synthesize relative risk or hazard ratios across clinical studies:

Code 
trials <- data.frame(
  Study = c("UK Haem Study (2022)", "EHA Registry (2024)", "ASH Consortium (2025)", "Blood Doctor Cohort (2026)", "Pooled Meta-Analysis"),
  HR = c(0.68, 0.72, 0.61, 0.58, 0.65),
  Lower = c(0.51, 0.58, 0.45, 0.42, 0.54),
  Upper = c(0.91, 0.89, 0.82, 0.80, 0.78),
  Summary = c(FALSE, FALSE, FALSE, FALSE, TRUE)
)

ggplot(trials, aes(x=HR, y=reorder(Study, c(5,4,3,2,1)))) +
  geom_vline(xintercept=1, linetype="dashed", color="grey40") +
  geom_errorbarh(aes(xmin=Lower, xmax=Upper), height=0.25, color="#7f0000", linewidth=1) +
  geom_point(aes(shape=Summary, size=Summary, fill=Summary), color="#7f0000") +
  scale_shape_manual(values=c(21, 23)) +
  scale_size_manual(values=c(3.5, 5)) +
  scale_fill_manual(values=c("#7f0000", "#ffd5d5")) +
  labs(title="Forest Plot of Treatment Hazard Ratios",
       x="Hazard Ratio (95% CI)", y=NULL) +
  theme_minimal(base_size=12) +
  theme(legend.position="none", axis.text.y=element_text(face="bold"))

3. Python engine ({python}) — pharmacokinetics & a pandas table

The same workflow speaks Python. This is the source that models one-compartment IV-bolus kinetics and builds a pandas summary table:

import numpy as np, matplotlib.pyplot as plt, pandas as pd

dose, Vd, t_half = 1000.0, 50.0, 12.0
ke = np.log(2) / t_half
t  = np.linspace(0, 48, 200)
C  = (dose / Vd) * np.exp(-ke * t)

fig, ax = plt.subplots(figsize=(7, 3.8))
ax.plot(t, C, color="#B71C1C", lw=2.2, label="Concentration C(t)")
ax.axhline(5, ls="--", color="grey", label="MEC (5 mg/L)")
ax.fill_between(t, C, 5, where=(C > 5), alpha=0.15, color="#B71C1C")
ax.set(xlabel="Time (h)", ylabel="Plasma concentration (mg/L)")
ax.legend(); ax.grid(alpha=0.3); plt.show()

## 1. R Engine (`{r}`): Kaplan-Meier Survival Curve

Using R and `ggplot2`, we generate a publication-ready Kaplan-Meier survival curve for clinical trials:

```{r}
#| label: fig-survival
#| fig-cap: "Kaplan-Meier overall survival curve generated dynamically with R and ggplot2."
#| fig-width: 7
#| fig-height: 4.2
library(ggplot2)

set.seed(2026)
time <- seq(0, 36, by=1)
surv_control <- exp(-0.06 * time) + rnorm(length(time), 0, 0.01)
surv_treated <- exp(-0.025 * time) + rnorm(length(time), 0, 0.01)

df <- data.frame(
  Time = rep(time, 2),
  Survival = c(pmax(0, surv_control), pmax(0, surv_treated)),
  Arm = factor(rep(c("Standard Care", "Targeted Therapy"), each=length(time)))
)

ggplot(df, aes(x=Time, y=Survival, color=Arm, linetype=Arm)) +
  geom_line(linewidth=1.2) +
  scale_color_manual(values=c("#7f0000", "#2e7d32")) +
  scale_y_continuous(labels=scales::percent, limits=c(0, 1)) +
  labs(title="Overall Survival in Relapsed/Refractory Disease",
       subtitle="Kaplan-Meier Survival Estimates by Treatment Arm",
```

## 2. R Engine (`{r}`): Meta-Analysis Forest Plot

```{r}
#| label: fig-forest

trials <- data.frame(
```

## 3. Python engine (`{python}`) — pharmacokinetics & a `pandas` table

```python
import numpy as np, matplotlib.pyplot as plt, pandas as pd

dose, Vd, t_half = 1000.0, 50.0, 12.0
```
